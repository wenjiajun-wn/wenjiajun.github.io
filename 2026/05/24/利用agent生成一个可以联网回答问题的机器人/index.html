基于agent生成一个联网回答问题的机器人

首先就是导入好需要的模块

import io
import os
import requests
from dotenv import load_dotenv
sys.stdout = io.TextIOWrapper(sys.stdout.buffer, encoding='utf-8')```
其中sys,io,os是python中的标准库，用于系统操作，文件路径，输入输出等等
requests：用于发送HTTP的请求，本质上是用来调用百度搜索的API
dotenv：是读取.env中的环境变量，就是API密钥
sys.stdout：这个是为了强制把控制台输出编码设置为UTF-8


```from langchain_openai import ChatOpenAI
from langchain.agents import create_agent
from langchain.tools import tool```
ChatOpenAI：是用来接入OpenAI 格式的 API
create_agent：是LangChain 1.0 中创建智能体的函数
tool：是把普通函数变为LangChain的可用工具，是一个装饰器

```load_dotenv()```
是从.env文件中读取密钥的东西

两个工具：calculator(数学计算表达式)和web_search(通过百度 AI 搜索 API 获取实时信息)，其中LangChain 的 create_agent，负责调用模型和工具，具体流程就是用户问“上海天气如何” → Agent 把问题发给 DeepSeek → 模型根据 system_prompt 判断需要搜索 → 调用     web_search 工具 → 百度 API 返回天气信息 → 模型组织回答 → 用户收到结果

其中有些注意事项：
1、百度的API每天只有100次的免费额度
2、假如模型不调用工具时可以在命令行更改强制调用工具，也可以调整system_prompt或者更换更强力的模型
3、.env中的密钥在自己上传代码前需要忽略掉避免自己的API泄露

怎么成功忽略? 
第一步先初始化Git仓库： git init
第二步： 创建 .gitignore 文件(一定是在根目录下创建)，然后再里面添加
``` .env
   __pycache__/
   *.pyc
   .vscode/
   .idea/```
第三部：创建 .env.example 文件，其中要包含
     DEEPSEEK_API_KEY=your_deepseek_api_key_here
     BAIDU_API_KEY=your_baidu_api_key_here
